## Docker Compose

- 각 디렉토리에 접속하여 아래와 같은 명령어로 Container 를 생성할 수 있습니다.

```
### 명령어 실행을 즉시 실행하는 경우
docker-compose up

### 명령어 실행을 백그라운드 프로세스로 실행하는 경우
docker-compose up -d
```
